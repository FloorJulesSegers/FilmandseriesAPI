 ## Fetching the API 
I used [Axios](https://github.com/axios/axios) to fetch the API.

```
axios.get('URL + input value + API_key')
  .then(function (response) {
    // handle success
    console.log(response);
  })
  .catch(function (error) {
    // handle error
    console.log(error);
  })
  .finally(function () {
    // always executed
  });
```
